# 🔨 페르소나 대장간 (ST-Persona-Forge)

SillyTavern용 **{{user}} 페르소나 프로필 자동 생성** 확장 프로그램입니다.

챗봇의 캐릭터 디스크립션과 월드인포를 분석하여, 해당 챗봇에 어울리는 {{user}} 페르소나 프로필을 AI가 자동으로 작성해줍니다.

## ✨ 주요 기능

### 🎯 스마트 생성
- **자유 생성**: 캐릭터 정보만으로 AI가 자유롭게 어울리는 페르소나 생성
- **가이드 생성**: "이런 캐릭터를 만들고 싶다" 컨셉을 입력하면 맞춤 생성

### 📖 참조 데이터
- 캐릭터 디스크립션, 성격, 시나리오 등 자동 참조
- 월드인포/로어북 엔트리 선택적 포함 (지시문 필터링 지원)

### 📝 유연한 템플릿
- **Basic**: 핵심 정보 (기본 정보, 외모, 성격, 관계)
- **Standard**: 표준 프로필 (+ 배경, 말투)
- **Detailed**: 상세 프로필 (+ 능력, 습관, AI 가이드라인)
- **Full**: 전체 프로필 (+ NSFW, 성적 취향, 숨겨진 욕망)
- **Choice**: 원하는 필드만 직접 선택
- **Custom**: 원하는 캐릭터 프로필 시트를 직접 입력
- 
### 🔄 편집 & 재생성
- 전체 재생성 또는 특정 섹션만 선택 재생성
- 재생성 시 추가 지시사항 입력 가능 ("좀 더 쿨하게", "키를 작게" 등)
- 직접 편집 모드 지원

### 🌐 다국어
- 생성 언어: English / 한국어 / 日本語 / 中文
- 생성 완료 후 다른 언어로 번역 가능 (의미 유지)

### 🔗 API
- SillyTavern Connection Profile 선택 가능
- 현재 연결된 API 바로 사용 가능

### 💾 저장
- 클립보드 복사
- ST 페르소나에 직접 저장
- 히스토리 저장/관리

## 🔧 사용 방법

1. ⚒️ 마법봉 확장 메뉴에서 **페르소나 대장간** 열기
2. 📌 대상 캐릭터 선택
3. 📖 (선택) 월드인포 엔트리 선택
4. 🎯 생성 모드 선택 (자유/가이드)
5. 📝 템플릿 선택
6. 🌐 출력 언어 선택
7. ⚒️ **페르소나 생성** 클릭!
8. 결과 확인 → 필요시 섹션별 재생성/편집
9. 💾 복사 또는 페르소나에 적용
